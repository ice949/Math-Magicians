- JavaScript, CSS
- React
- Github, Linters

## Live Demo

[Math Magicians](https://ice949.github.io/Math-Magicians/)

## GETTING STARTED

1. Open Terminal

### Run

2.  ```
    git clone https://github.com/ice949/Math-Magicians.git
    ```

3.  ```
    cd Math-Magicians
    ```

4.  ```
    npm start
    ```
